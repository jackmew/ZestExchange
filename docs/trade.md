# Recent Trades 的顏色代表「誰是主動方 (Taker)」，這在交易界是標準的做法。

  這決定了這筆交易被視為「買入」還是「賣出」：

   * 綠色 (Buy)：代表 主動買入 (Taker Buy)。
       * 意義：有人不願意排隊，直接用高價「吃掉」了掛在牆上的賣單 (Asks)。
       * 市場解讀：這通常被視為買盤強勁，有人在搶貨。
   * 紅色 (Sell)：代表 主動賣出 (Taker Sell)。
       * 意義：有人不願意排隊，直接用低價「砸掉」了掛在牆上的買單 (Bids)。
       * 市場解讀：這通常被視為賣壓沉重，有人在逃命。

  為什麼不看 Maker？
  因為每一筆成交一定同時有一個買家和一個賣家。如果只看成交，你分不出它是好是壞。所以業界統一觀察 Taker
  (主動發起者) 的方向：
   * 如果 Taker 是買方，這筆成交就是綠色。
   * 如果 Taker 是賣方，這筆成交就是紅色。

  這能讓交易者一眼看出現在市場是被「多頭 (Aggressive Buyers)」主導，還是「空頭 (Aggressive
  Sellers)」主導。
